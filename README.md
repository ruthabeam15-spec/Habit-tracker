# Habit Tracker
A simple and clean React Habit Tracker that allows you to add habits, mark them as completed, and now delete them, with all data saved automatically using localStorage.

Built with React and Vite, this project uses a clean and easy to use interface.

Features:
Add new habits
Mark habits as completed (green highlight + strikethrough)
Delete habits using a remove button
Habits save automatically via localStorage
Clean and minimal user interface
